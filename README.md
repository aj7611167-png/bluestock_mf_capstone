# Bluestock Mutual Fund Analytics Capstone

## Project Overview

This capstone project analyzes mutual fund performance, investor behaviour, SIP continuity, portfolio concentration, and risk metrics using Python, SQL, and Tableau.

Features completed:

* ETL Pipeline
* Data Cleaning
* Exploratory Data Analysis
* Dashboard Development
* Risk Analytics (VaR / CVaR)
* Rolling Sharpe Analysis
* Investor Analytics
* Fund Recommendation System

---

## Objectives

* Extract and process mutual fund data
* Perform data validation
* Analyze fund performance
* Build investor insights
* Create interactive dashboards
* Compute advanced risk metrics
* Generate recommendations
